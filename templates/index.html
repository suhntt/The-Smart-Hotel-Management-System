System Workflow

How The System Works

1. Registration & Booking

Users sign up, select available rooms from the database, and make a simulated secure payment.

2. Document Upload

Customers are required to upload their Aadhaar card image and a clear selfie for security records.

3. Reception Scan

The admin interface uses OCR and face recognition to compare physical cards against uploaded records.

4. System Sync

If verified, the room status changes to Occupied and the customer receives a digital pass.
